本工程基于Unity 5.0/4.6.4 + uLua 1.1.0 + cstolua构建
服务器端基于VS2012及其以上版本。

支持平台：PC/MAC/Android(armv7-a + Intel x86)/iOS(armv7 + arm64)/
	  WP8(SimpleFramework_WP_v0.1.1 (nlua))/
